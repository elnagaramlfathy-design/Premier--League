# Premier League Data Analysis

A comprehensive data analysis project exploring English Premier League statistics. 

## Project Overview
This project focuses on analyzing team performance, player insights, and match outcomes. The analysis was conducted using Microsoft Excel and Google Sheets to create interactive dashboards.

## Key Features
- Data cleaning and preparation.
- Performance metrics analysis.
- Visualization of key trends through dashboards.

## Files Included
- Team 5 (1).zip: Contains the full dataset and the final dashboard files.

## Tools Used
- Microsoft Excel
- Google Sheets
